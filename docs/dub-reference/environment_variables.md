# Environment Variables

## Environment variables DUB reads

- DUB_HOME
- DPATH
- DFLAGS
- `<PKG>_VERSION`
- http_proxy
- no_proxy
- NO_COLOR
- TEMP (Windows only)
- DUB_NO_RESOLVE_LIMIT
- DUB_REGISTRY
- DUBPATH
- DC
- PATH
- ProgramData (Windows only)
- APPDATA (Windows only)
- HOME (POSIX only)
- USERNAME (Windows only)
- USER (POSIX only)
